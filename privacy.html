Privacy Policy

Trello ID Tools Power-Up

Overview

Trello ID Tools is a Trello Power-Up that adds configurable card buttons for copying Trello IDs, card URLs, and structured metadata to your clipboard. This policy describes how the Power-Up handles data.

Data Collection

This Power-Up does not collect, store, or transmit any personal data. No analytics, tracking, or logging of any kind is performed. No data is sent to any external server or third-party service.

Clipboard Access

When you click a copy button, the Power-Up reads the relevant Trello value (such as a card ID or URL) from the Trello API and writes it directly to your clipboard using the browser's standard Clipboard API. This value is never stored, logged, or transmitted anywhere.

Settings Storage

Button preferences (which copy buttons are shown) are saved using Trello's built-in board-level plugin data storage. This data stays entirely within Trello's own infrastructure and is never accessed by any external party.

Third-Party Services

This Power-Up is hosted on GitHub Pages as a static site. No server-side code runs. No cookies are set. No user information is processed outside of the Trello iframe context.

Contact

If you have questions about this privacy policy, please open an issue on the GitHub repository.
